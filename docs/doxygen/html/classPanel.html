Detailed Description

Abstract base class for all UI panels.

Panels provide modular UI components rendered using Dear ImGui. Each panel has read/write access to the simulation state and camera, allowing interactive control and visualization of simulation parameters.

Panels are managed by PanelManager and rendered in the main UI layout.
