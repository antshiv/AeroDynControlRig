Detailed Description

UI panel displaying rotor performance metrics.

Shows for each rotor:

  • RPM (revolutions per minute)
  • Thrust (Newtons)
  • Torque (Newton-meters)
  • Total thrust and power consumption

Visualizes data from RotorTelemetryModule using gauges, bar charts, or numeric readouts.
