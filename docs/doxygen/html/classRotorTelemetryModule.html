Detailed Description

Computes rotor thrust, torque, and power from RPM measurements.

This module calculates performance metrics for a quadcopter's rotors using first-principles aerodynamics:

Thrust: F = k_T * ω² Torque: τ = k_Q * ω² Power: P = τ * ω

Where:

  • ω = angular velocity (rad/s) = RPM * (2π/60)
  • k_T = thrust coefficient (N/(rad/s)²)
  • k_Q = torque coefficient (N·m/(rad/s)²)

The module currently generates synthetic RPM data (sinusoidal variation). Future enhancements:

  • Accept commanded RPM from controller module
  • Add rotor dynamics (lag between command and actual RPM)
  • Include motor voltage/current modeling
