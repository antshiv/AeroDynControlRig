Detailed Description

UI panel displaying simulated IMU sensor measurements.

Shows:

  • Gyroscope readings (rad/s) in body frame
  • Accelerometer readings (m/sÂ²) in body frame

Data is generated by SensorSimulatorModule and displayed as numeric values or gauges.
