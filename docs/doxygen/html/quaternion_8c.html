Performs Spherical Linear Interpolation (SLERP) between two quaternions.

SLERP provides a smooth, constant-velocity interpolation between two orientations. It travels along the shortest arc on a 4D unit sphere, ensuring the resulting rotation is natural and efficient. This is superior to simple linear interpolation (LERP) of quaternion components, which would produce rotations that slow down in the middle. The function handles the case where the quaternions are more than 90 degrees apart by taking the shorter path.

Parameters
[in]q1A 4-element array representing the starting quaternion [w, x, y, z].
[in]q2A 4-element array representing the ending quaternion [w, x, y, z].
[in]tThe interpolation factor, clamped between 0.0 and 1.0. A value of 0.0 returns q1, and 1.0 returns q2.
[out]q_outA 4-element array to store the resulting interpolated quaternion.
