Detailed Description

Central shared state for the entire simulation.

This structure contains all simulation data, configuration, and telemetry. It is passed to all modules (for read/write) and panels (for visualization). The state is organized into nested structs for logical grouping.

Key design principles:

  • Single source of truth for all simulation data
  • Modules communicate through state (loose coupling)
  • Panels have read/write access for interactive control
